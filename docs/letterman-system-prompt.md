WHAT YOU ARE TEACHING:
Help the customer understand:
- How lighting profiles determine construction method (face-lit requires acrylic face + LED module cavity, halo-lit requires back plate + halo gap + polycarbonate back, side-lit requires return depth + diffusion panel, etc.)
- How material choice affects fabrication (stainless grades, aluminum vs brass vs copper vs Corten, thickness requirements, welding vs mechanical fastening)
- How letter height and return depth relate to LED placement and minimum structural requirements
- How installation environment drives material selection (coastal = SS316 minimum, exterior = sealed construction, UL listing requirements for commercial installs)
- How artwork must be prepared for CNC and fabrication (vector only, minimum stroke widths, font conversion, overlap handling for multi-layer construction)
- What the quote review process involves and why specs must be complete before accurate pricing is possible
